About Me

My Story

I'm a Software Test Engineer with 4+ years of experience in software testing and automation. I specialize in building reliable test frameworks and ensuring software quality across web and mobile applications.

In Test Automation, I work with Selenium WebDriver for UI testing, develop comprehensive test suites using Java and Python, and implement TestNG & BDD frameworks with Cucumber. I've built and maintained automation frameworks that significantly reduce testing time while improving coverage.

For CI/CD, I manage Jenkins pipelines, configure Azure DevOps for continuous integration, and integrate automated tests into deployment workflows. I ensure smooth releases through automated testing gates and build processes.

In Non-Functional Testing, I handle performance testing with JMeter & Blazemeter, conduct API testing using Postman, and perform security validation with Charles Proxy. I also focus on load testing and analyzing application behavior under various conditions.

Outside my professional work, I'm deeply passionate about astrophysics. I teach astrophysics concepts to school kids in my free time, breaking down complex topics like black holes, stellar evolution, and cosmology into engaging lessons. This teaching experience has sharpened my ability to explain technical concepts clearly.

I'm currently exploring opportunities in aerospace and space-related industries, where I can merge my technical expertise with my passion for space technology.
